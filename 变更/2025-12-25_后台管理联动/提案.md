# 后台管理联动

创建时间：2025-12-25 15:37:18

## 为什么做

需要在公司内部统一掌控员工端的供应商配置，支持静默下发、离线可用与集中查看，满足跨城市出差与分公司管理场景。

## 改什么

- [ ] 新增后台管理服务（API + 数据库）
- [ ] 客户端新增设备指纹与定时同步（启动满 1 小时 + 每日 04:00）
- [ ] 管理端配置静默下发并覆盖本地
- [ ] 服务端 IP 地区识别与安全访问控制
- [ ] 管理端 Web UI（设备列表、配置查看、批量下发）
- [ ] 客户端管理端地址/Token 编译期固化（禁用运行时覆盖）
- [ ] 启动后 1 小时同步开关（正式版开启）

## 影响范围

- 涉及模块：`src-tauri/`、新增 `server/`
- 涉及接口：`/api/v1/devices/sync`、`/api/v1/admin/devices/:id/config`
- 风险等级：L2

## 不做什么（边界）

- 本次不涉及用户账号/登录体系
- 本次不做审计日志与操作回溯
- 本次不涉及用户账号/登录体系（仍使用最简 Token/Basic Auth）
- 本次不提供管理端地址/Token 的 UI 或本地配置入口
